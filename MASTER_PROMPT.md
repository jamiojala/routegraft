# routegraft

Build a serious open source TypeScript package for AI middleware.

Repository name: routegraft
npm package: @jamiojala/routegraft

Core concept:
AI middleware for Express, Hono, Fastify, and Next route handlers.

Requirements:
- TypeScript
- pnpm
- Vitest
- tsup or tsdown
- Changesets
- GitHub Actions
- strong docs and examples

Deliverables:
- package.json
- src
- tests
- examples
- docs
- README.md
- CONTRIBUTING.md
- LICENSE
- CI workflow

Goal:
Production middleware for AI routes and streams, with streaming helpers, quotas, policy hooks, and clean developer ergonomics.
